# Put them to work 💬

In **Copilot Chat**, type `@army` followed by any task:

```
@army find where the navbar is rendered
@army add a dark-mode toggle
@army fix the crash on startup
```

The **Captain** reads the mission, routes it to the right specialist, and answers
in that agent's voice — using its persona from `.github/agents/`.

Two built-in commands:

- `@army /roster` — list the whole army
- `@army /recruit` — scaffold a brand-new agent
