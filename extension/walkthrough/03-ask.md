# Put them to work 💬

In **Copilot Chat**, type `@squad` followed by any task:

```
@squad find where the navbar is rendered
@squad add a dark-mode toggle
@squad fix the crash on startup
```

The **Captain** reads the mission, routes it to the right specialist, and answers
in that agent's voice — using its persona from `.github/agents/`.

Two built-in commands:

- `@squad /roster` — list the whole squad
- `@squad /recruit` — scaffold a brand-new agent
