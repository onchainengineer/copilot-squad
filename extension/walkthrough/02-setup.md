# Deploy the army 🚀

**Set Up the Army** writes the full army into your workspace `.github/` folder:

```
.github/
├─ copilot-instructions.md   ← the army's shared charter
├─ agents/                   ← 6 custom Copilot agents (.agent.md)
└─ prompts/                  ← reusable skills (.prompt.md)
```

These are real GitHub Copilot customization files. Once deployed, the agents show
up in Copilot Chat's agent picker — and in the **Army** sidebar on the left.

Open `.github/copilot-instructions.md` afterwards and tailor it to *your* project.
